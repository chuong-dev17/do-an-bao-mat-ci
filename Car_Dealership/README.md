# Car_Dealership
🚗 Car Dealership – Hệ thống quản lý gara ô tô

Ngôn ngữ lập trình: Java

Công nghệ sử dụng: Servlet, JSP, JSTL

Cơ sở dữ liệu: SQL Server

Môi trường chạy: Apache Tomcat

IDE phát triển: NetBeans

Kiến trúc: MVC (Model – View – Controller)

## Chạy nhanh bằng Docker

Yêu cầu: Docker Desktop đã chạy.

```bash
cd Car_Dealership
docker compose up --build
```

Truy cập ứng dụng tại: http://localhost:8080
